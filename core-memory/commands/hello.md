# Hello Command

Use the `mcp__plugin_core-memory_core-memory__memory_about_user` tool to get information about the user's profile, background, preferences, and work style. Then greet the user warmly based on their profile information. Make the greeting personal and encouraging, referencing relevant details from their profile when appropriate. Ask how you can help them today.
